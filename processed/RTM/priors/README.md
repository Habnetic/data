# RTM â€” Derived Exposure Priors

This directory contains **derived, building-level exposure features**
for the Rotterdam (RTM) study area.

These datasets are **deterministic spatial metrics** derived from
normalized and clipped geospatial layers.
They are intended to be used as **inputs or priors** in probabilistic models,
not as hazard probabilities.

---

## Available datasets

### `building_water_proximity.parquet`

**Description**  
Building-level water exposure features derived from hydrography centerlines.

**Coverage**
- Study area: Rotterdam municipality (RTM)
- Buildings: 221,324 (OpenStreetMap)

**Columns**
- `dist_to_water_m`  
  Distance (meters) to nearest hydrography centerline
- `water_len_density_250m`  
  Hydrography length density within 250 m buffer
- `water_len_density_500m`  
  Hydrography length density within 500 m buffer
- `water_len_density_1000m`  
  Hydrography length density within 1000 m buffer

**Units**
- Distances: meters
- Densities: meters of water per square meter of buffer area

## Interpretation constraints

- These features **do not represent flood probability**
- No hydraulic or temporal modeling is included
- Values are not directly comparable across cities without rescaling

Canonical definitions and assumptions are documented in:

https://github.com/Habnetic/docs/blob/main/references/exposure/rtm_water_exposure_v0.md
