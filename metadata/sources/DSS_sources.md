# Donostia–San Sebastián (DSS) — Data Sources

**Focus:** coastal and pluvial flooding, wind exposure, urban resilience.

| Domain | Source | Dataset | Format | License | Notes |
|:--|:--|:--|:--|:--|:--|
| Climate | Copernicus Climate Data Store | ERA5 Reanalysis (precip, temp) | NetCDF | CC-BY 4.0 | Baseline forcing data |
| Hazards | Basque Government Geoportal | Flood hazard zones | GeoPackage | ODC-By | Regional layer |
| Hazards | Copernicus Emergency Mgmt | Flood maps | GeoTIFF | CC-BY 4.0 | Sentinel-1 derived |
| Buildings | OpenStreetMap | Building footprints | GeoJSON | ODbL | Filter by admin boundary |
| Socio | EU-STATGRID / INE | Population grid | CSV / GeoTIFF | CC-BY 4.0 | 1 km resolution |
